## ğŸš€ Installation Guide

### Step 1: Clone or Create Project Structure

```bash
mkdir sports-booking-platform
cd sports-booking-platform
mkdir backend frontend
```

### Step 2: Backend Setup

```bash
cd backend
npm init -y
```

**Install Backend Dependencies:**
```bash
npm install express mysql2 sequelize dotenv bcryptjs jsonwebtoken cors
npm install --save-dev nodemon
```

**Create MySQL Database:**
```sql
mysql -u root -p
CREATE DATABASE sports_booking;
EXIT;
```

**Create .env file in backend folder:**
```env
PORT=
DB_HOST=
DB_USER=
DB_PASSWORD=
DB_NAME=
JWT_SECRET=
NODE_ENV=
```

**Start Backend:**
```bash
# Seed database with test data
npm run seed

# Start development server
npm run dev
```

Backend should now be running on `http://localhost:5000`

---

### Step 3: Frontend Setup

```bash
cd ../frontend
npm create vite@latest . -- --template react
```

**Install Frontend Dependencies:**
```bash
npm install
npm install react-router-dom axios date-fns
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

**Create .env file in frontend folder:**
```env
VITE_API_URL=http://localhost:5000/api
```

**Start Frontend:**
```bash
npm run dev
```

Frontend should now be running on `http://localhost:3000`

## ğŸš¨ Common Issues & Solutions

### Backend Issues

**Database Connection Failed**
```bash
# Check MySQL is running
sudo service mysql status

# Verify credentials in .env
# Try connecting manually:
mysql -u root -p sports_booking
```

**Port Already in Use**
```bash
# Kill process on port 5000
lsof -ti:5000 | xargs kill -9

# Or change PORT in .env
```

**Seeding Fails**
```bash
# Drop and recreate database
mysql -u root -p
DROP DATABASE sports_booking;
CREATE DATABASE sports_booking;
EXIT;

# Run seed again
npm run seed
```

### Frontend Issues

**API Calls Fail**
- Check backend is running on port 5000
- Verify VITE_API_URL in .env
- Check browser console for errors
- Verify no CORS issues

**Styles Not Showing**
```bash
# Reinstall Tailwind
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p

# Restart dev server
npm run dev
```

**Token Expired**
- Tokens last 30 days
- Simply login again
- Clear localStorage if needed
